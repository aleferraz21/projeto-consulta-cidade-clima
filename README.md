# API de Consultas De Cidades e Climas

Trabalho de Integração de Sistemas para consulta de cidades e clima.

## Como rodar o projeto
1. Certifique-se de ter o Node.js instalado.
2. No terminal, rode: `npm install`
3. Para iniciar o servidor: `npm start`
4. Para rodar os testes: `npm test`

## Tecnologias utilizadas
- Node.js / Express
- Axios / Jest / Supertest
- Postman para testes
